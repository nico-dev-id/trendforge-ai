# ADR-001

Title:
Use TypeScript Instead of JavaScript

Status:
Accepted

Date:
2026-06-07

Context:

Project bertujuan meningkatkan peluang kerja remote.

Decision:

Frontend menggunakan TypeScript.

Reason:

TypeScript lebih banyak digunakan di industri.

Consequences:

Perlu waktu belajar tambahan.
Code lebih aman.
